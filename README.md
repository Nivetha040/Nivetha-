Speech-to-Text Converter

This project is a simple and efficient Speech-to-Text converter that captures spoken audio and converts it into written text using speech recognition libraries.

Features

Real-time speech recognition

Supports multiple languages (configurable)

Microphone-based input

Option to save transcripts to a file

Lightweight and easy to use


Technologies Used

Python 3.x

SpeechRecognition

PyAudio

Google Web Speech API (default recognizer)


Installation

1. Clone the repository:



git clone https://github.com/your-username/speech-to-text.git
cd speech-to-text

2. Install dependencies:



pip install -r requirements.txt

3. (Optional) Install PyAudio if not included:



pip install pyaudio

If you face issues on Windows, use:

pip install pipwin
pipwin install pyaudio

Usage

Run the Python script:

python speech_to_text.py

Start speaking and the recognized text will appear in your terminal or be saved to a file.

Output

Transcribed text is printed in the terminal

Optionally saved to transcript.txt


Configuration

Change language: Modify language='en-US' in the script to support different languages (e.g., hi-IN for Hindi)

Adjust microphone settings if needed
